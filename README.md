# AI Job Portal 🚀

## Features
- Student & Recruiter Authentication
- Job Posting & Application System
- Resume Upload (PDF)
- AI-based Resume Matching (TF-IDF + Cosine Similarity)
- Applicant Ranking System
- Dashboard for Students & Recruiters

## Tech Stack
- Frontend: React.js
- Backend: Django REST Framework
- Database: PostgreSQL
- AI: Scikit-learn (TF-IDF)

## How to Run

### Backend
cd backend
pip install -r requirements.txt
python manage.py runserver

### Frontend
cd frontend
npm install
npm start
